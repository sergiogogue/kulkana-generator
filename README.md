## 🚀 INSTALACIÓN:

### 1. Descargar
Descarga `kulkana-epic.zip`

### 2. Descomprimir
Extrae todo el contenido

### 3. Desplegar en Netlify
- Ve a https://app.netlify.com/drop  
- Arrastra la CARPETA completa descomprimida
- Netlify detectará automáticamente las Functions

### 4. Configurar API Key
- Abre la URL que te da Netlify
- Click en 🔑 (esquina superior derecha)
- Pega tu API Key de Anthropic
- ¡Listo!

## 📋 CÓMO USAR:

1. **Sube PDF** → Arrastra ficha técnica
2. **Extrae** → Click "Extraer Datos"
3. **Completa** → Revisa y llena campos faltantes
4. **Genera** → Click "Generar Presentación"
5. **Descarga** → Tu PPTX está listo
